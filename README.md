# Test-assignment-for-an-internship

Алгоритм для вычисления минимальной общей сетевой маски (CIDR) между двумя произвольными IP-адресами.

## Реализованные возможности

- **Поддержка протоколов:** IPv4 и IPv6 (Усложнение A).
- **Валидация (Усложнение B):** - Проверка корректности диапазонов (0-255 для IPv4, 0-FFFF для IPv6)
  - Запрет ведущих нулей в IPv4 (защита от неверной интерпретации чисел)
  - Корректное развертывание сокращенной записи IPv6 (`::`)
  - Проверка на недопустимые символы и количетсво сегментов
- **Обработка ошибок:** В коде реализованы сообщения об ошибках. При вводе пустых строк, смешивании протоколов или нарушении структуры адреса, программа вызовет исключение ValueError с подробным описанием причины

## Технологии

- **Язык:** Python
- **Библиотеки:** Не используются

## Примеры работы

На вход функции get_min_mask подаются два ip адреса: 192.168.1.10, 192.168.1.20  
Результат содержит в себе версию ip, prefix и саму маску: IPv4, /27, 255.255.255.224

## Инструкция по запуску

1. Запустить скрипт test_task_for_job.py
2. Программа выполнит встроенные тесты (assert) и выведет результаты для пар адресов
